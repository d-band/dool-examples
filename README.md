# Examples

> All examples changed based on [spm-example](https://github.com/spmjs/examples/tree/spm-webpack)

Project examples for `dool` .

## Prepare

```bash
$ npm install dool -g
```

## Usage

```bash
# !-- 1. enter directory
$ cd react

# !-- 2. debug
$ dool server
$ open http://127.0.0.1:8000/

# !-- 3. build
$ dool build
```
